Randomiser Mod

The Randomiser turns Fear & Hunger into a chaotic rogue-like experience. It shuffles enemy troops, items, weapons, armors, and optionally the dungeon maps themselves, creating a different run each time you play. A seed system lets you share or replay specific randomisations.

What Gets Randomised

CategoryWhat Changes
Troops / Encounters Random encounters are pulled from a shuffled pool of all defined troops. You may encounter bosses where you would normally find basic enemies, and vice versa.
Items Item identities are shuffled. A "Torch" in the world might give you a sword. Pick up everything — you never know what it is until you check.
Weapons & Armors Equipment loot tables are reshuffled independently of general items.
Maps (optional) When the Dungeon Knight map generation is enabled, procedural maps replace all dungeon floors. Each floor is a 50×50 grid of rooms, corridors, and encounters.

Seeds

Every randomised run is driven by a numeric seed. The same seed always produces the same shuffled run on the same version of the mod.

How Seeds Work

  • If you start a randomised run without specifying a seed, a random seed is generated and stored with the game.
  • The seed is saved in save files, so loading a save will always continue the same randomised run.
  • To share a seed with a friend, find it in the randomiser menu or the dev tools overlay, then tell them the number.

Sharing Seeds

Share a seed in a place like Discord to let friends or streamers replicate your exact experience. Both players must use the same mod version for the seed to produce identical results.

Using with Multiplayer

The Randomiser is fully compatible with the Multiplayer mod. The seed is distributed to all clients by the host at session start, so all players experience the same randomisation.

If a client joins mid-run with a different seed state than the host, there may be desyncs. It is best to start the randomised run from a fresh New Game in multiplayer.
