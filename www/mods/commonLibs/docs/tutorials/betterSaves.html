Better Saves

The Better Saves mod overhauls Fear & Hunger's default save system. It removes the tight vanilla save slot cap, adds a Suspended Run (Save and Quit) feature, and improves the save/load UI so managing long or multiple playthroughs is much more comfortable.

Expanded Save Slots

Vanilla Fear & Hunger limits the number of save slots you can have. Better Saves raises this cap significantly, defaulting to 99 save slots. The limit is configurable in the mod settings.

All extra slots behave identically to the vanilla save slots — they appear in the save/load menu and are stored as standard .rpgsave files in the save/ folder.

Suspended Run (Save and Quit)

What Is It?

The Suspended Run feature lets you save your game to a special suspend slot and quit immediately, without filling one of your regular save slots. When you come back, you can continue from the exact moment you suspended — including your position on the map, all inventory, and all game state.

This is useful for sessions where you need to stop playing mid-dungeon without losing progress, or when you want a clean "I can always pick this back up" save without committing it to a numbered slot.

How to Use It

  1. Open the in-game menu.
  2. You will see a "Save and Quit" (or "Suspend Run") option.
  3. Select it. The game saves to the suspend slot and returns you to the title screen.
  4. The next time you launch, a "Continue Suspended Run" button appears on the main menu if a suspended save exists.
  5. Click it to resume exactly where you left off.

Notes

  • Only one suspended run exists at a time. Starting a new suspension overwrites the previous one.
  • The suspend slot is stored as save slot 9998 internally (file9998.rpgsave). It does not appear in the regular save/load list.
  • The "Continue Suspended Run" button only appears on the main menu when a suspended save file actually exists.

Compatibility

Better Saves is compatible with the Multiplayer mod. However, in multiplayer sessions only the host's save file is the canonical game state. Clients should not rely on their own save files as the source of truth during a multiplayer run.

Better Saves is also compatible with the Battle Auto-Backup system — battle snapshots are created for all active save slots regardless of which Better Saves features you use.
