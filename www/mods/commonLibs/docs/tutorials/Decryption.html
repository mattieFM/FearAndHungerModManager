How to decrypt your game files

  • What is Decryption

    In fear and hunger & RPGMaker MV decryption has 2 parts:

    1. Converting encrypted files like .rpgmvp and .rpgmv to unencrypted files, like .png or .jpg. This can be done with various tools and web options.
    2. Updating the settings.json file to let the game know weather it is dealing with encrypted or unencrypted files. This can be done by opening the settings.json file and making sure "hasEncryptedFiles" is set to "false". And is automatically updated by some decryption tools.
